Design Responsivo

  1. Introdução

    2.   O conceito de Web Design Responsivo ou Design Responsivo, que é relativamente
    novo, surgiu em 2010, onde o artigo “Responsive Web Design” foi publicado pelo
    desenvolvedor Ethan Marcotte, onde ele citava que a forma que os usuários acessavam
    aos serviços da internet estava mudando, e para acompanhar essa mudança, todos os sites
    e aplicações deviam acompanhar e se adaptar a essa mudança, com isso empresas e
    negócios tiveram que adaptar seus sites aos mais diversos dispositivos, com diversas resoluções.

  2. Como surgiu?

    3.   Web Design Responsivo ou Design Responsivo é um conceito ou técnica relacionado a
    interface de sites ou aplicações, sendo a ação que faz com que uma página de internet ou a
    interface de uma aplicação se adapte com o dispositivo utilizado pelo usuário ao acessá-lo,
    podendo ser um smartphone, tablet ou computador, o objetivo dessa técnica é fazer o site se
    adaptar com o dispositivo do usuário, com o intuito de fazer com que o site ou aplicação seja
    acessível a todos os usuários

  3. Importância do Design Responsivo

    4.   Com a constante evolução dos dispositivos móveis e dos computadores, e maior
    utilização dos mesmos, é necessário que as empresas ou negócios que possuam seus sites
    e/ou aplicações, adaptem os mesmos com a técnica do Design Responsivo, com o intuito
    de atender a todos os seus clientes, facilitando o acesso de cada usuário com seu
    determinado dispositivo. Também existem os benefícios de se ter apenas uma versão do
    site, onde você precisara atualizar ele apenas uma vez, tendo em vista que ele vai se
    adaptar a todos os dispositivos, também terá um gerenciamento melhor, em relação a
    atualizações.

    Características do Design Responsivo

  1. Media Queries

    2.   As Media Queries, que são um recurso do CSS 3, onde ele consulta as especificações do dispositivo de um usuário,
    possibilitando assim a adaptação de um site ou aplicação ao dispositivo utilizado pelo usuário.

  2. Grids Fluídos

    3.   Um site com um grid fluído se adapta a resolução de qualquer dispositivo utilizado, sendo um smartphone ou computador,
    adaptando o seu conteúdo para que ele caiba na tela do dispositivo do usuário, para que o conteúdo não fique muito pequeno
    ou muito grande.

  3. Layouts Fluídos

    4.   É uma técnica onde a visualização do conteúdo de um site ou aplicação se adapta ao dispositivo do usuário, podendo ter suas informações
    mostradas de formas diferentes, como se você tivesse um site ou aplicação, mas com diversas versões, correspondentes as diversas resoluções
    de telas, para que seja algo natural.

  4. Mobile First

    5.   É um conceito apresentado por Luke Wroblemski, em seu livro Mobile First (WROBLEWSKI,2011), onde o conceito apresentado
    se refere a iniciar a criação de um site pequeno até um grande sistema, por exemplo, começando o projeto de um site pela sua
    versão mobile, isso estaria preparando você para o crescimento explosivo da utilização de celulares e outras tecnologias.

    Conclusão

      Web Design Responsivo ou Design Responsivo, como é mais conhecido, é uma técnica relativamente nova e altamente
    importante nos tempos atuais, onde o crescimento da tecnologia está presente, estão sendo criadas os mais diferentes
    dispositivos que auxiliam no nosso cotidiano, seja um smartphone novo, um computador ou um tablet, esses dispositivos
    estão em constante evolução, com isso está a importância de se ter um site ou uma aplicação que se adapte as diferentes
    resoluções de telas. Para isso existe o Design Responsivo, que é uma técnica que consiste em criar a interface do seu site
    ou aplicação, com diferentes combinações de técnicas, para que ele se molde de acordo com o dispositivo utilizado pelo
    usuário, também é importante mencionar que é importante que empresas ou negócios que possuam algum site ou aplicação,
    esteja atualizando os mesmos, para garantir que seus clientes consigam acessá-los sem ter algum problema, para
    não ter problema algum com sua clientela.
